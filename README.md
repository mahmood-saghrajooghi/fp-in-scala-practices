# FP in Scala Practice

This project is a small learning playground for functional programming in Scala, based on exercises from *Functional Programming in Scala*.

It currently includes:
- A custom immutable `List` data type
- Core list operations (`foldLeft`, `foldRight`, `sum`, `product`, `reverse`, etc.)
- Simple examples in `main` (`factorial`, `fib`, and list operations)

## Run

```bash
scala-cli run main.scala
```
